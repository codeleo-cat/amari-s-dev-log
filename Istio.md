# Ingress controller 역할을 수행

- service 보호, 연결, 모니터링
- 통신에 대한 측정

**Service Mesh**
- Application에 추가할 수 있는 전용 인프라 계층

**Traffic Control**
- virtual service : 어느 host 혹은 service로 routing 할지, 트래픽(weight)를 어느 정도로 할지
- gateway : 어느 host의 요청, 포트 등을 처리할 지
