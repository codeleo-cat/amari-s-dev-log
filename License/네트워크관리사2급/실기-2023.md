### Router

- show process
- ip add `ip address` & ip add `ip address` secondary
- 활성화 O -> no shutdown

- 도메인 ICQA로 설정 **ip domain-name**
```
en
conf t
ip domain-name ICQA
exit
copy r s
``` 

- dhcp pool **ip dhcp pool & network**
```
en
conf t
ip dhcp pool icqa
network 192.168.100.0 255.255.255.0
exit
exit
copy r s
```

en
